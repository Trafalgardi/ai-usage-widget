Frequently asked questions

What does the widget track?

It displays Claude Code and Codex CLI session limits, weekly quotas, reset countdowns and account status when exposed by the services.

Is this an API billing tracker?

No. It focuses on subscription and CLI usage limits rather than API token costs.

Does it work on Windows 10 and Windows 11?

Yes. It is designed for current Windows 10 and Windows 11 installations with WebView2 available.

Is it affiliated with Anthropic or OpenAI?

No. AI Usage Widget is an independent open-source project.
